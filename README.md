# 🚀 FranchiseHub Landing Page

A modern and responsive landing page for a franchise business built using HTML, CSS, and JavaScript.

## 🔥 Features

- Fully Responsive Design (Mobile + Desktop)
- Modern UI with Gradient Theme
- Interactive Navbar with Mobile Menu
- Smooth Scrolling Navigation
- Active Section Highlight on Scroll
- Clean and Structured Layout
- Contact Form UI

## 📂 Sections Included

- Hero Section
- About Section
- Franchise Model
- Benefits
- Contact Form
- Footer

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
